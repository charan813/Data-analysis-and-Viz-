# Data-analysis-and-Viz-
Contains data analysis and visualizations using numpy, pandas, matplotlib and seaborn packages 
##Basketball_stats contains the following
### Matrices:
### - Salary
### - Games
### - MinutesPlayed
### - FieldGoals
### - FieldGoalAttempts
### - Points
## Lists:
### - Players
### - Seasons
## Dictionaries:
### - Sdict 
### - Pdict

## Comments:  
### Seasons are labeled based on the first year in the season   
### E.g. the 2012-2013 season is presented as simply 2012
